# Distinctions

## Against Deleuze

- Deleuze holds that difference is primary; this work holds that 
  the first act of difference is contingent, not structural.
- Deleuze's difference operates as a system; this work addresses 
  the moment before any system is possible.
- Deleuze presupposes difference to explain difference; this work 
  names that presupposition as the structural limit.

## Against Spirin (2025)

- Spirin grounds differentiation in Potentiality; this work holds 
  that Potentiality is itself a distinction and cannot precede 
  the first distinction.
- Spirin's pre-ontological condition is still a condition — a 
  relation between what could differentiate and what has not; 
  this work finds no relation before the first crack.
- Spirin explains the first differentiation; this work holds that 
  explanation here is impossible in principle.

## Against Prior Works in This Series

- Structure of Reality establishes that objects are 
  crystallizations of gaps; this work establishes that the first 
  gap has no origin.
- Holding Time establishes that Holding has no direction and no 
  accumulation; this work establishes that it also has no reason 
  to end.
- Holding the Gap provides a protocol for sustaining the gap; 
  this work shows that the first gap was never sustained — 
  it simply was, until it was not.
